# 功能说明

## 1. 计划页

计划页是整个应用的核心工作区，用于查看某一天的学习安排、调整任务状态，并结合课程表与规则进行排程。

### 1.1 今日学习计划

- 展示当前所选日期的全部任务
- 支持切换日期查看不同天的任务列表
- 支持折叠和展开
- 折叠后仍显示当前最优先任务摘要

任务卡片中会展示：

- 任务标题
- 科目
- 优先级
- 状态
- 安排模式
- 开始时间
- 截止时间
- 预计时长
- 已专注时长

如果任务是从前一天延期而来，还会显示“由某天延期而来”的提示。

### 1.2 任务状态

每个任务支持以下状态切换：

- 未开始
- 进行中
- 已完成
- 延期

规则说明：

- 点击“延期”后，任务会移动到次日
- 灵活任务会按规则自动寻找次日合适时间段
- 延期任务在第二天仍保留延期来源信息
- 如果把延期任务改回“未开始”，会视为取消延期并返回原日期
- 如果把延期任务改成“进行中”或“已完成”，会保留在当前日期，不会错误跳回原日期

### 1.3 新增任务

支持填写：

- 任务名
- 科目
- 优先级
- 预计时长
- 开始时间
- 截止时间
- 安排模式

安排模式：

- 固定时间
- 允许自动改期

在“允许自动改期”模式下：

- 系统会实时计算可用建议时段
- 点击添加任务时，如果手动时间不满足规则，会自动改用建议时段

### 1.4 建议排程

点击“建议排程”后，系统会基于以下条件自动调整任务顺序和时间：

- 高优先级优先安排
- 中优先级随后安排
- 低优先级最后安排
- 避开课程时间
- 遵守可安排时间区间
- 遵守优先时间区间
- 遵守当前北京时间最早开始限制
- 遵守课程前后缓冲间隔

## 2. 可视化时间轴

时间轴将课程和学习任务放在同一条时间轴上显示。

特点：

- 课程用蓝色显示
- 不同任务使用不同色彩
- 支持悬停显示详情
- 便于快速观察一天安排是否拥挤、是否冲突

## 3. 课程表模式

课程表模式允许用户按星期设置固定课程。

支持信息：

- 课程名称
- 星期几
- 开始时间
- 结束时间
- 地点

课程会参与：

- 自动排程避让
- 时间轴展示
- 任务冲突检测

## 4. 计划规则设置

该区域位于课程表模式下方、账号与同步上方，支持折叠。

可设置内容：

- 任务可安排的最早开始时间
- 任务可安排的最晚结束时间
- 3 组优先时间区间
- 临近课程前后的缓冲分钟数

这些规则会同时影响：

- 新增任务自动修正
- 建议排程
- 延期到次日后的自动重排

## 5. 番茄钟

番茄钟用于日常专注学习。

支持：

- 设置专注时长
- 设置休息时长
- 开始、暂停、重置
- 自动在专注和休息阶段间切换
- 浏览器通知提醒

稳定性说明：

- 切换页面不会停止计时
- 熄屏后恢复时会按真实经过时间结算

## 6. 考试页

考试页是适合电脑全屏使用的倒计时模式。

### 6.1 基本能力

- 设置考试内容
- 设置科目
- 设置小时和分钟
- 显示剩余时间 `HH:MM:SS`
- 显示总时长、已用时间、剩余时间、状态
- 显示考试进度条

### 6.2 控制按钮

- 开始考试
- 暂停
- 继续
- 重置
- 重新设置考试时间

### 6.3 交互行为

- 未开始时显示设置区
- 开始考试后隐藏设置区，只保留核心考试信息
- 进入全屏后只显示黑色考试面板核心内容
- 考试结束后自动退出全屏

### 6.4 提醒机制

在以下时间点会触发明显提醒：

- 剩余 30 分钟
- 剩余 15 分钟
- 剩余 5 分钟
- 剩余 1 分钟
- 考试结束

提醒形式：

- 页面提示
- 页面闪烁
- 声音提醒
- 浏览器通知

### 6.5 数据统计

考试完成后，本场考试会被记录为一条考试记录，并计入：

- 最近 7 天学习时长
- 各科目学习占比
- 本周学习时长

## 7. 统计页

统计页用于复盘近期学习表现。

包含：

- 最近 7 天学习时长
- 各科目学习占比
- 完成率趋势

统计来源包括：

- 已完成任务的预计/实际学习时间
- 番茄钟累计专注时间
- 考试模式累计时长

## 8. 提醒中心

支持浏览器通知授权和提醒触发。

提醒包括：

- 计划开始提醒
- 番茄钟结束提醒
- 晚上 21:30 复盘提醒
- 考试关键时间提醒

## 9. 账号与同步

支持使用 Supabase 进行登录和同步。

功能包括：

- 注册
- 登录
- 退出登录
- 云端快照同步

未启用 Supabase 时：

- 自动退回本地 `localStorage`
- 不影响本地使用

已启用 Supabase 时：

- 可在多设备间同步任务、课程、规则、考试和主题设置

## 10. 数据保存机制

### 本地保存

默认通过浏览器 `localStorage` 保存：

- 任务
- 课程表
- 番茄钟设置
- 考试计时状态
- 统计相关数据
- 主题和规则设置

### 云端同步

启用 Supabase 后，会将上述主要数据作为用户快照同步到云端。

## 11. 缓存与兼容性

为了减少浏览器缓存导致的旧版本冲突，项目已经做了两类处理：

- `index.html` 不长期缓存
- 静态资源使用带哈希文件名

同时还针对 Safari 做了兼容优化。
